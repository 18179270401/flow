微信公众号设置

  • 公众号Appid

    登录微信公众号平台：http://mp.weixin.qq.com，进入《基础设置》页面，将获取到公众号AppId，填写在此处, AppId又称为应用ID,为微信公众号唯一标志.营销场景中。用于获取参加流量活动的用户信息。方便企业汇聚用户信息。查看用户分布。查找用户习性。

  • 公众号AppSecret

    登录微信公众号平台：http://mp.weixin.qq.com，进入《基础设置》页面，将获取到公众号AppSecret，填写在此处, AppSecret又称为应用密钥。为匹配AppId所用，微信提供安全性验证。

  • 微信号

    登录微信公众号平台：http://mp.weixin.qq.com，进入《公众号设置》页面，将获取到公众号微信号，填写在此处，营销场景中。可以根据微信号。给用户提供微信公众号二维码。从而让用户达到关注的目的。
