流量场景>微信授权平台

第三方服务平台接入。

爱讯流量平台将接管微信公众号以下功能：
1.消息管理权限：帮助公众号接收用户信息，进行自动回复权限。
2.网页服务权限：帮助公众号实现第三方网页服务和活动。
3.用户管理权限：帮助公众号获取用户名信息，进行用户管理。

请企业使用微信公众号管理员的微信对生成的授权二维码进行授权。

用户将公众号授权给爱讯流量之后。
1.企业可以控制自己公众号对用户的自定义回复。
2.企业可以用对话的模式让用户参与活动。

授权须知：

微信公众号授权须知以下几点：
1.用户开通订阅号授权时。需要在微信设置中填写appid。否则会出现。授权失败的现象。
2.如果用户需要取消授权。需要在公众号内部取消。取消授权后。订阅号服务管理会失效。
3.如果用户没有填写微信设置。进行公众号授权。造成授权失败。请填写appid。然后再在公众号后台取消授权后。再次授权。
