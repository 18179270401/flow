开发者账号

账号名称(account)和Key值(apikey)在哪获取？

用户登录平台，在导航菜单流量中心-开发者中心-开发者账号下面。对接api主要参数就是这2个，请保证账号的安全性。

回调地址在哪配置？

在导航菜单流量中心-开发者中心-开发者账号下面。回调地址主要是用户需要平台主动发送订单最终信息给用户自己的终端。

IP鉴权地址怎么填写？

有需要对用户接口发送的服务器进行ip限制，就可以使用ip鉴权进行配置。多个ip使用英文,号分隔。配置空或者255.255.255.255表示不鉴权。

签名sign的加密格式是什么？

例如：apiKey+account=XXXXX&action=Charge&phone=XXXXX&range=0&size=10&timeStamp=********+apiKey。
